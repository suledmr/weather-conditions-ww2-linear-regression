# Weather Conditions in World War II - EDA and Linear Regression

This project explores weather conditions during World War II using exploratory data analysis and a Linear Regression model.

## Project Overview

The goal of this project is to analyze weather-related data and build a regression model to understand relationships between weather variables.

## Project Steps

- Data loading
- Data cleaning
- Exploratory Data Analysis
- Data visualization
- Feature selection
- Linear Regression model training
- Model evaluation

## Dataset

Dataset source: Kaggle
Dataset link: <https://www.kaggle.com/datasets/smid80/weatherww2>

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
